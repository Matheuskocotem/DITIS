Início Rápido

Para começar a usar a API DITIS, siga estes passos:

  1. Faça login através do endpoint /api/login para obter um token
  2. Inclua o token no header Authorization: Bearer {token} nas requisições
  3. Consulte a documentação da API para detalhes de cada endpoint
  4. Veja exemplos práticos para integração rápida
